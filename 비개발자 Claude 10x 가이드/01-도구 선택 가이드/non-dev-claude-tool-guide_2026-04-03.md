# 비개발자를 위한 Claude 도구 선택 가이드 — Chat vs Cowork vs Code, 직무별 매핑

> 최종 수정: 2026-04-03

## TL;DR

Claude에는 세 가지 모드가 있다: Chat(대화), Cowork(자율 실행), Code(개발). 비개발자는 Chat + Cowork 조합으로 대부분의 업무를 자동화할 수 있다. 직무별로 어떤 도구가 맞는지 매핑하면 툴 선택으로 고민하는 시간을 없앨 수 있다.

---

## Part 1: 세 가지 도구의 차이

### 한 줄 정의

| 도구 | 한 줄 정의 | 비개발자 접근성 |
|------|-----------|--------------|
| **Claude Chat** | 대화형 AI 어시스턴트 | ⭐⭐⭐⭐⭐ 누구나 |
| **Claude Cowork** | 자율 실행 에이전트 (파일 읽기/쓰기/웹검색/스케줄) | ⭐⭐⭐⭐ 클릭만 가능하면 OK |
| **Claude Code** | 개발자용 코드 에이전트 (터미널 기반) | ⭐⭐ 개발자 전용 |

### 상세 비교

**Claude Chat**
- 질문하고 답 받는 기본 모드
- 웹 검색 내장 (최신 정보 가능)
- 파일 접근 불가, 백그라운드 실행 불가
- 적합한 작업: 아이디어 구상, 글쓰기, 리서치, 요약, 번역

**Claude Cowork**
- 다단계 작업을 자율로 실행하는 에이전트 모드
- 로컬 폴더에 직접 접근 → 파일 읽기/쓰기/생성
- 웹 검색 + Python 실행 + 외부 서비스 연동 (Gmail, Notion, Slack 등)
- 스케줄링: 매주/매일 자동 실행 가능
- 적합한 작업: 반복 업무 자동화, 대량 파일 처리, 보고서 생성

**Claude Code**
- 터미널(CLI) 기반 개발 에이전트
- 코드베이스 전체를 읽고 수정
- 비개발자는 사용 불가 (진입장벽 높음)
- 적합한 작업: 코드 작성, 버그 수정, 기능 구현

### 언제 뭘 쓸까 — 판단 기준

```
질문/아이디어/빠른 답변이 필요하다    → Chat
파일을 만들거나 자동화하고 싶다       → Cowork
코드를 짜거나 수정해야 한다           → Code (개발자만)
```

## Part 2: 직무별 도구 매핑

### 매트릭스

| 직무 | 주력 도구 | 보조 도구 | 주요 활용 |
|------|----------|----------|---------|
| 마케터 | Cowork | Chat | 콘텐츠 자동화, SNS 멀티포맷 생성 |
| 기획자/PM | Chat | Cowork | PRD 작성, 리서치, 회의록 정리 |
| HR | Cowork | Chat | 채용공고, 온보딩 문서, 계약서 초안 |
| 영업 | Cowork | Chat | 제안서, 이메일 개인화, CRM 정리 |
| 재무/경영지원 | Cowork | Chat | 지출 분류, 월간 보고서, 엑셀 자동화 |
| 운영/총무 | Cowork | - | 파일 정리, 반복 문서, 일정 요약 |

### 직무별 구체적 활용

#### 마케터
**Chat 활용**
- 캠페인 아이디어 브레인스토밍
- 카피라이팅 초안 (페르소나 기반)
- 경쟁사 분석 (웹 검색 활용)

**Cowork 활용**
- 블로그 1편 → SNS 5개 채널 포맷으로 자동 변환
- 트렌드 키워드 매주 자동 수집 → 마크다운 파일 저장
- 월간 콘텐츠 캘린더 자동 생성
- Before/After: 콘텐츠 제작 월 4편 → 12편

#### 기획자/PM
**Chat 활용**
- PRD(제품 요구사항 문서) 초안 작성
- 유저 스토리, 플로우 설계
- 회의 전 리서치, 데이터 수집

**Cowork 활용**
- 회의록 파일 → 액션 아이템 추출 → 담당자별 이메일 초안 자동 생성
- 다수의 인터뷰 녹취록 → 공통 인사이트 정리
- Before/After: 회의록 정리 2시간 → 15분

#### HR
**Chat 활용**
- 채용공고 초안 작성
- 면접 질문 리스트 생성
- 인사 정책 문서 검토

**Cowork 활용**
- 내부 JD + 회사 톤앤매너 → 채용공고 자동 완성
- 온보딩 문서 배치 생성 (신규 입사자마다)
- Before/After: 채용공고 작성 3시간 → 20분

#### 영업
**Chat 활용**
- 고객사 사전 리서치
- 제안서 논리 구조 설계
- 이메일 초안 작성

**Cowork 활용**
- 고객 정보 파일 → 개인화 제안서 자동 생성
- 이메일 응답률 개선: 일반 이메일 → 고객별 맞춤 레퍼런스 포함
- Before/After: 이메일 응답률 8% → 23%

#### 재무/경영지원
**Cowork 활용**
- 카드 명세서 → Excel 지출 분류 + 요약
- 월간 보고서 자동 생성
- Before/After: 경비 처리 4시간 → 10분

## Part 3: 도구 선택 플로우차트

```
내가 하고 싶은 일이 뭔가?
│
├─ 아이디어/질문/글쓰기/번역
│   └─ → Claude Chat
│
├─ 파일을 만들거나 반복 작업을 자동화하고 싶다
│   ├─ 한 번만 하면 됨 → Claude Chat (파일 첨부 후 처리)
│   └─ 매주/매일 반복 → Claude Cowork (스케줄 등록)
│
└─ 코드를 직접 짜야 함
    └─ 개발자 있음 → Claude Code (개발자에게 넘기기)
        개발자 없음 → Cowork로 Python 자동 생성 후 실행
```

## Part 4: 시작하기 — 비개발자 세팅 가이드

### Claude Chat (즉시 시작)
1. claude.ai 접속 또는 Claude Desktop 설치
2. 무료 플랜으로 시작 가능
3. 프롬프트 입력 → 바로 사용

### Claude Cowork (5분 세팅)
1. Claude Desktop 다운로드 (Mac/Windows)
2. 유료 플랜 필요 (Pro $20/월 추천)
3. 앱 실행 → Cowork 탭 클릭
4. "Work in a folder" → 작업 폴더 지정
5. 할 일을 자연어로 설명 → 실행

### 글로벌 설정 (선택)
- Settings → Cowork → Global instructions
- 회사 톤앤매너, 출력 형식, 역할 배경 입력
- 예: "항상 한국어로 작성, 회사명은 XX, 격식체 사용"

### 외부 서비스 연동 (선택)
- Connectors에서 원클릭 인증
- Gmail, Google Drive, Notion, Slack, Calendar 지원
- 연동 후 "Gmail에서 오늘 받은 메일 요약해줘" 바로 가능

## 핵심 인사이트

1. **도구 선택 원칙 하나**: "파일이 필요하면 Cowork, 대화면 Chat" — 이 기준 하나로 80%는 해결된다.
2. **비개발자의 진짜 레버리지는 Cowork**: 반복 업무를 스케줄로 등록하는 순간 10x가 시작된다. 코딩 없이 명세서(자연어)만으로 Python 자동화까지 가능.
3. **직무별 최고 ROI**: 마케터(콘텐츠 3배 생산), HR(채용공고 9배 빠르게), 영업(응답률 3배), 재무(경비처리 24배 빠르게) — 반복 문서가 많을수록 효과가 극대화된다.
