# React + TypeScript 프로젝트

이 프로젝트는 React + TypeScript 기반입니다.
사용자가 /init 호출 시, 이 파일과 아래 규칙 파일들을 기반으로 `.claude/CLAUDE.md`를 생성하세요.

## 규칙 매핑

- 컴포넌트 생성/수정 시 → `components.md`
- 파일, 변수, 함수 이름 지을 때 → `naming.md`
- 스타일/CSS/TailwindCSS 작업 시 → `styling.md`
- 설계 판단이 필요할 때 → `design-principles.md`
- 인터랙티브 UI 작업 시 → `accessibility.md`
- 폼 구현 시 → `form.md`
- API 호출/데이터 패칭 시 → `api.md`
- 보안 관련 코드 작성 시 → `security.md`
- 성능 최적화 필요 시 → `performance.md`
- 코드 리뷰/셀프 체크 시 → `code-quality.md`
- 테스트 작성 시 → `testing.md`
- 폴더 구조/파일 배치 시 → `structure.md`
- 에러 처리 구현 시 → `error-handling.md`
